# SARTHAK – Smart Aerial Response and Tracking for Health And Kin Safety

SARTHAK is an **emergency response and monitoring drone platform** designed to help identify humans, monitor movement patterns, and mark locations on a live map for rapid response operations.

The system combines aerial mobility, camera sensing, and onboard electronics to assist in **search, monitoring, and safety applications**.

SARTHAK focuses on creating a **low-cost aerial monitoring system** capable of supporting emergency response teams with faster situational awareness.

---

## Overview

SARTHAK is a **hexacopter drone platform** built for aerial monitoring and response support.

The drone can:

- Detect human presence using onboard vision systems
- Monitor movement patterns
- Mark detected locations on a live map
- Assist responders with aerial situational awareness

The platform integrates a camera system and monitoring electronics into a stable multirotor drone capable of carrying sensors and surveillance equipment.

## System Concept

SARTHAK operates as an **aerial monitoring drone** designed to support emergency response.

Typical operation flow:

1. The drone flies over a monitoring area.
2. The onboard camera observes the environment.
3. Detection algorithms identify humans or movement patterns.
4. Detected locations are marked on a monitoring map.
5. Emergency responders can use this information for faster action.

This system helps provide **real-time situational awareness in emergency environments**.
